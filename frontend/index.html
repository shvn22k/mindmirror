Upload a short face video and call the local inference API. Start the backend with uvicorn backend.main:app --port 8000 from the project root.

Live webcam

Records short segments (same idea as a clip), sends each to /predict, and stores outcomes for export. Use HTTPS or localhost so the browser allows the camera. If WebM fails on the server, install OpenCV with FFmpeg support or use Chrome and ensure codecs are available.
